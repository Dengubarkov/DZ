Правила использования системы генерации пароля:

<--На главную
  1. Система генерирует пароль согласно установленным параметрам
  2. Перед созданием пароля нужно определить параметры длины и установить сложность через подключение дополнительных символов разметки*
  3. После выбора необходимых параметров нужно нажать кнопку "Сгенерировать пароль"
* По умолчанию система использует только английские буквы в нижнем регистре
